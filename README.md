# EcpayLogistics

綠界物流串接用 SDK（ecpay_logistics-1.0.8.gem）

From: https://github.com/ECPay/Logistic_RoR

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ecpay_logistics'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ecpay_logistics
